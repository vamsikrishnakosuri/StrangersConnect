# Logo Instructions

## Adding Your Logo

1. **Save your logo file** as `logo.png` in this `public/` directory
2. **Recommended size**: 
   - At least 512x512px for best quality
   - PNG format with transparent background (or white background)
   - The logo will automatically scale to fit

## Logo Placement

The logo will be used in:
- **Header**: Next to the app name (top left)
- **Homepage**: Large logo display when not matched
- **Favicon**: Browser tab icon

## Fallback

If the logo file is not found, the app will automatically fall back to:
- Emoji icon (üé•) in the header
- Globe emoji (üåê) on the homepage

## File Format

- **Format**: PNG (recommended) or SVG
- **Name**: `logo.png` or `logo.svg`
- **Location**: `/public/logo.png`

Your logo will be accessible at: `https://yourdomain.com/logo.png`
